# Node.js Express Recipe App
A full-stack web application built using Node.js, Express, Pug, and MySQL.

## Features
- Dynamic routing with Express
- Server-side rendering with Pug
- MySQL database integration
- User account creation
- Contact form with email functionality (Nodemailer)

## Technologies
- Node.js
- Express.js
- Pug
- MySQL

## Database Setup
This project requires a MySQL database.

- Create a database named `recipecentral`
- Ensure the database connection settings in `server.js` match your local setup

## How to Run
1. Install dependencies:
   npm install

2. Run the server:
   npm start

3. Open browser:
   http://localhost:7000
